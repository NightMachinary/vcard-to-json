#+TITLE: vcard-to-json/readme

* build
#+begin_example zsh
clj -T:build rebuild-uber
#: outputs in ./target/
#+end_example

* usage
#+begin_example zsh
cat my_vcard |
  java -jar ./target/vcard-to-json*.jar
#+end_example

Or you can use it without building the uberjar:
#+begin_example zsh
cat my_vcard |
  clj -X NightMachinery.vcard-to-json
#+end_example
